The OS X blog

I’ve never kept it a secret that I’m a huge fan of OS X. I love this OS! So I’ve set a category aside for my notes and thoughts on OS X in general. Yes, I work for Mandrake, and yes, I use Mandrake. But I also use OS X (in fact, I use OS X primarily for my desktop systems). I prefer Mandrake as a server distro, however, I’m also liking Annvix for servers as well. Linux has it’s place on the desktop, don’t get me wrong, but for my own personal gratification, I like OS X better. That’s not to say that one day Linux may surpass OS X, but to me, it hasn’t hit that critical mass yet.

Anyways, here I will rant about OS X.. the good, the bad, the ugly.
