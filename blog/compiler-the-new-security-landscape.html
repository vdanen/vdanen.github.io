Compiler: The New Security Landscape

Tomorrow, the first episode of the new Compiler podcast season is coming out, and this season is all about product security topics! The season’s title is “The New Security Landscape” and we dive into quite a few topics pertinent to security practitioners (or simply the curious!). I don’t think I’m allowed to say what all the topics are yet so you’ll need to stay tuned!

You can get to the Compiler podcast on YouTube or go to the main Compiler podcast site for other ways to listen.

Unlike my Security Unscripted podcast, Compiler is just audio. And I get to co-host this with my new bestie, Emily Bock. We tried really hard to ensure that the topics don’t woosh over most heads, and had quite a bit of fun doing it as well.

I’m really excited about this project and am grateful for the opportunity to be a co-host. I love the fact that we have an entire season devoted to product security. In the meantime, if you’re interested in technical topics, check out prior seasons and episodes as there’s some really cool topics that have been discussed in the past.
